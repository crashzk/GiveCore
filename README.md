# GiveCore

Модуль, расширяющий Flute поддержкой выдачи различных привилегий с условиями. Не имеет интерфейса, нужен для других модулей (Shop, Vip и т.д.)

Module that extends Flute with support for issuing various privileges with conditions. Does not have an interface, needed for other modules (Shop, Vip, etc.)

## Installation

Download the latest release and install it via the Flute CMS admin panel.

Current version: **2.2.3**
